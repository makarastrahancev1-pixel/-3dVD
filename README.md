## 🎮 Описание

**Базука 3D** — это браузерная 3D-игра, в которой игрок управляет персонажем с базукой. Можно разрушать здания, стрелять по NPC, вызывать метеориты и общаться с искусственным интеллектом через чат. Игра поддерживает как ПК (клавиатура + мышь), так и мобильные устройства (сенсорное управление).

---

## 🕹️ Управление

### ПК (клавиатура + мышь)

| Клавиша | Действие |
|---------|----------|
| `W, A, S, D` | Передвижение |
| `Мышь` | Обзор |
| `ЛКМ` | Выстрел из базуки |
| `R` | Перезарядка |
| `G` | Режим бога (полёт) |
| `H` | Расширение территории (бег x6) |
| `E` | Вызов метеорита |
| `F` | Спавн людей (в спецрежимах) |
| `T` | Восстановить все здания |
| `P` | Открыть чат |
| `Пробел` | Вверх (в режиме бога) |
| `Shift` | Вниз (в режиме бога) |
| `⋮` (кнопка) | Меню паузы |

### Телефон (сенсорное управление)

| Элемент | Действие |
|---------|----------|
| `Джойстик` (слева) | Передвижение |
| `Проведение по экрану` | Обзор |
| `Кнопка 🔫` | Выстрел |
| `Кнопка ⬆️` | Прыжок/вверх (в режиме бога) |
| `Кнопка ☄️` | Метеорит |
| `Кнопка 👑` | Переключение режимов |

## 🌟 Режимы игры

### Обычный режим
- Скорость: **1x**
- Патроны: **ограничены** (5 шт.)
- Взрывы: **стандартные**
- Передвижение: **по земле**

### 👑 Режим бога
- Скорость: **2x**
- Патроны: **бесконечные**
- Взрывы: **x1.3**
- Передвижение: **полёт**
- Активация: клавиша `G`

### ⭐ Расширение территории
- Скорость: **6x** (только бег)
- Патроны: **бесконечные**
- Взрывы: **x1.5**
- Особенность: **пентаграмма** под ногами
- Активация: клавиша `H`
- Спавн людей: **x3** больше

## ✨ Особенности

- 🏙️ **Разрушаемый город** — здания разрушаются от взрывов
- 🧱 **Физика объектов** — ящики разлетаются при взрывах
- 👥 **NPC с ИИ** — люди ходят по городу, умирают от атак
- 💬 **Умный чат** — NPC отвечают на ключевые слова
- ☄️ **Метеориты** — падают с неба по команде
- 💾 **Сохранение прогресса** — через localStorage
- 📱 **Адаптивный дизайн** — работает на ПК и телефонах
- 🎯 **Прицел** — для точной стрельбы
- 🔄 **Восстановление** — кнопка сброса всех разрушений

## 💾 Система сохранения

### Как сохранить:
1. Нажмите кнопку `⋮` (в правом верхнем углу)
2. Выберите `💾 СОХРАНИТЬ`
3. Прогресс сохранится в браузере

### Как загрузить:
1. В главном меню нажмите `▶️ ПРОДОЛЖИТЬ`
2. Игра загрузится с места сохранения

### Что сохраняется:
- Позиция игрока
- Количество патронов
- Активные режимы (бог/территория)
- Состояние зданий (разрушены/целы)
- Направление взгляда

## 💬 Чат с NPC

### Как открыть:
- **ПК**: клавиша `P`
- **Телефон**: кнопка чата (в разработке)

### Ключевые слова для ответов NPC:

| Категория | Ключевые слова |
|-----------|----------------|
| Приветствие | `привет`, `здарова` |

| Оружие | `базука`, `оружие` |
| События | `взрыв`, `метеорит` |
| Город | `город`, `здания` |
| Режимы | `бог`, `пентаграмма` |

| Благодарность | `спасибо`, `благодарю` |

## 🔧 Технические детали

### Используемые технологии:
- **Three.js** — 3D-графика
- **Cannon-es** — физический движок (в ранних версиях)
- **LocalStorage** — сохранение прогресса
- **Touch Events** — мобильное управление
- **Pointer Lock API** — захват мыши

### Оптимизации:
- Адаптивное количество объектов (меньше на телефонах)
- Низкополигональные модели (картофельная графика)
- Ограничение частиц взрывов
- Fog (туман) для скрытия дальних объектов
- LOD-подобная система (меньше деталей на мобильных)

### Производительность:
- **ПК**: 60 FPS (стабильно)
- **Телефон**: 30-50 FPS (зависит от устройства)

## 🐛 Известные баги

1. При очень большом количестве взрывов возможны просадки FPS
2. На слабых телефонах может нагреваться корпус
3. Сохранение не работает в приватном режиме браузера
4. Иногда NPC застревают в зданиях

---

## 💡 Советы

- **Для максимального FPS**: используйте режим "картофельной графики" (уже включён для телефонов)
- **Для веселья**: включите режим бога и вызывайте метеориты
- **Для челленджа**: играйте в обычном режиме с 5 патронами
- **Для общения**: используйте чат с NPC — они отвечают на 100+ фраз
